# 更改 collection 的路径

```sh
[defaults]
inventory = ./hosts
collections_paths = ./ansible_collections:/root/.ansible/collections
```

![alt text](README_Images/安装/image-3.png)
![alt text](README_Images/安装/image-4.png)

# 验证配置是否生效（路径成功得到了更换)

```sh
ansible-config dump | grep COLLECTIONS_PATHS
```
