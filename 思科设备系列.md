---

### 📦 思科设备分类一览表

| 分类               | 系列                                                    | 主要用途                    | 备注                   |
| ------------------ | ------------------------------------------------------- | --------------------------- | ---------------------- |
| 🔌 接入交换机      | Catalyst 1000 / 2960-X / 9200 / Meraki MS120            | 用户接入、终端连接          | 支持 VLAN、PoE 等      |
| 🧠 核心/汇聚交换机 | Catalyst 9300 / 9400 / 9500 / 9600 / Meraki MS350-MS390 | 核心网络、汇聚层            | 支持堆叠、三层路由     |
| 🌐 路由器          | ISR 1000 / 4000, ASR 1000 / 9000, Meraki MX             | 分支机构 WAN 连接、核心路由 | 支持 MPLS、VPN 等      |
| 🔐 防火墙          | Cisco ASA / Firepower (FTD) / Meraki MX                 | 边界安全、VPN、防攻击       | 支持 IPS/IDS、VPN      |
| 📡 无线控制器      | Catalyst 9800 / Meraki MR + Dashboard                   | 无线 AP 集中管理            | 云管或本地部署         |
| 🛡️ 安全设备        | ISE / Firepower Management Center (FMC) / Umbrella      | 身份控制、安全策略          | NAC、DNS 保护          |
| 📊 管理平台        | Cisco DNA Center / Meraki Dashboard                     | 网络可视化、自动化管理      | 云管或本地私有云       |
| 📦 光模块          | SFP / SFP+ / QSFP                                       | 光纤连接、模块扩展          | 根据速率和设备型号选择 |
| 💽 存储与计算      | UCS (Unified Computing System)                          | 数据中心服务器              | 适合虚拟化、云平台     |
| ☁️ SD-WAN          | vEdge / cEdge / vSmart / vManage                        | 广域网优化、分支互联        | SDN 架构下的 WAN       |

---

### ✅ 温馨提示：

- 如果你要构建企业网络，通常组合为：
  - 接入层：Catalyst 9200/MS120
  - 汇聚层：Catalyst 9300/MS250+
  - 核心层：Catalyst 9500/9600
  - 安全：ASA / Firepower / Meraki MX
  - 无线：MR AP + 9800 控制器 或 Meraki 云控制器
