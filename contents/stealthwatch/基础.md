# Cisco stealthwathch 现在叫做 Cisco Secure Network Analytics 作用？

Cisco Stealthwatch（现为 **Cisco Secure Network Analytics**）的核心能力主要体现在 **网络可视化、威胁检测** 和 **取证分析** 三大方面。下面是它的具体核心能力：

---

### 1. **网络可视化（Network Visibility）**

- 通过采集**NetFlow、sFlow、IPFIX 等流量元数据**，对整个网络中的通信行为进行持续、全面的监控。
- 可以看到哪些设备在通信、通信了多少数据、使用了哪些协议和端口 —— 即使是加密的流量，也能分析其模式。
- 支持跨越私有网络、数据中心、云平台（如 AWS、Azure）等环境的统一可视化。

---

### 2. **行为分析与威胁检测（Behavioral Analytics & Threat Detection）**

- 使用**机器学习与行为建模**，识别异常行为，如数据泄露、横向移动、恶意通信等。
- 检测包括但不限于：
  - 内部威胁
  - 零日攻击
  - 勒索软件行为
  - 加密通信中的可疑行为（无需解密）
- 集成 Cisco Talos 威胁情报，提高检测精度。

---

### 3. **安全取证与溯源（Security Forensics & Investigation）**

- 提供完整的历史数据（可保留数月），支持事后分析和调查。
- 可以追踪攻击路径，分析攻击者从哪里来、攻击了哪些资产、用了什么方法。
- 取证信息可用于与防火墙、EDR、SIEM 系统联动。

---

### 4. **自动响应与集成能力（Automation & Integration）**

- 可与 Cisco ISE 联动，根据 Stealthwatch 的检测结果自动隔离可疑终端。
- 与 Cisco SecureX、Firepower、第三方系统（如 Splunk）集成，形成闭环防护。
- 提供 REST API 支持自动化操作和自定义集成。
