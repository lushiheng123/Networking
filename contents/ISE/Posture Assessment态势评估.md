# 什么是 ISE 态势评估(Posture Assessment)?

### 🔍 ISE 中的 Posture Assessment 是什么？

Cisco ISE 的 **Posture 服务**用于检查终端设备的合规性，比如：

- 是否安装了防病毒软件
- 是否开启了防火墙
- 操作系统是否打了补丁
- 等等……

ISE 可以根据设备的状态**动态决定是否允许它访问网络、是否隔离、或要求修复**。

---

### 🧠 ISE 中的 Posture Requirement 类型：

| 类型             | 说明                                                               |
| ---------------- | ------------------------------------------------------------------ |
| **Mandatory** ✅ | 强制要求修复。设备必须在规定时间内完成修复，否则会被限制访问。     |
| **Optional**     | 建议修复，但不强制，设备可以继续访问网络。                         |
| **Audit**        | 只检查记录，不做合规判断，也不限制访问。                           |
| **Visibility**   | 仅用于收集信息，不影响访问，也不显示不合规状态（适合做环境调查）。 |

---

### 举个简单例子：

你设定一个 **Mandatory Requirement**：

- 要求客户端必须开启防火墙
- 如果没开，就给它显示一个修复提示，并给它 2 分钟时间修复
- 如果 2 分钟后还没修复，就把它踢进隔离 VLAN
