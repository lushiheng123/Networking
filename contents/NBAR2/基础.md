NBAR2（**Network-Based Application Recognition version 2**）是 Cisco 提供的一项**深度应用识别技术**，用于识别经过网络的数据流中具体是哪一种应用。它是 NBAR 的升级版，支持更广泛、更智能的识别功能。

---

## 🔍 一句话定义：

> **NBAR2 是一种基于深度包检测（DPI）的智能应用识别技术，能帮助 Cisco 设备识别、分类、控制和统计网络流量中的具体应用。**

---

## 🚀 用途场景：

- **QoS 策略匹配：** 按应用类型进行优先级控制（如视频会议高优先）
- **流量分析与报告：** 与 NetFlow/Flexible NetFlow 结合，输出详细的应用统计数据
- **安全可视性：** 判断网络中是否有滥用应用（如 P2P、社交软件）
- **SD-WAN 路由决策：** 依据具体应用走不同的链路

---

## 🧠 NBAR2 可以识别哪些内容？

| 类型        | 举例                                     |
| ----------- | ---------------------------------------- |
| Web 应用    | YouTube、Netflix、Facebook、Zoom、Teams  |
| 协议        | HTTP, HTTPS, DNS, SIP, FTP, SSH          |
| SaaS 服务   | Salesforce, Office365, Dropbox           |
| P2P/影子 IT | BitTorrent、VPN 应用（如 HotspotShield） |
| 自定义应用  | 支持基于端口/IP/域名创建自定义分类       |

NBAR2 可以识别 **数千种应用**，其应用特征库是 **可自动更新的**（称为 **PDLM - Protocol Pack**）。

---

## 🔧 技术实现核心：

NBAR2 通过以下方式识别应用：

- 深度包检测（DPI）：检查 L7 协议头、payload 特征等
- 端口号 + IP + DNS 域名 + TLS SNI 字段 等多维度组合识别
- 使用 Protocol Pack 进行特征匹配（可定期更新）

---

## ✅ 支持 NBAR2 的设备示例：

- ISR 4000 系列
- ASR 1000 系列
- Catalyst 9000 系列交换机
- Cisco SD-WAN（vEdge, C-edge）
- 配合 NetFlow 或 Cisco DNA Center 查看报告

---

## 📌 举个例子：

你可以配置 NBAR2 来识别 Zoom、Teams、YouTube 等应用，然后用如下策略处理：

```bash
class-map match-any VIDEO-CONF
 match protocol zoom
 match protocol ms-teams

policy-map WAN-QoS
 class VIDEO-CONF
  priority percent 30
```

这样 Zoom 和 Teams 的流量就会被识别出来，并且设置为高优先级保障。
