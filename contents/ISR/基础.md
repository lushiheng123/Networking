# ISR 是什么？

<details>
<summary>GPT:cisco isr是什么？</summary>
Cisco ISR（**Integrated Services Router**，集成服务路由器）是 Cisco 推出的一系列**融合多种网络功能**的企业级路由器，专门为**分支机构、远程站点和中小型企业**设计。

---

## 🧩 一句话解释：

> **ISR 就是集“路由、防火墙、VPN、语音、安全、无线”等多种功能于一身的一体化网络设备。**

---

## 🔧 为什么叫“集成服务”？

因为它不是一个“只会路由”的设备，它可以同时完成多个角色：

| 功能模块             | 说明                                            |
| -------------------- | ----------------------------------------------- |
| **Routing（路由）**  | 支持静态路由、动态路由（如 OSPF、EIGRP、BGP）   |
| **Security（安全）** | 集成防火墙、IPS、VPN（IPSec/GRE/DMVPN/FlexVPN） |
| **WAN Optimization** | 可加载 WAAS 模块，提高广域网传输效率            |
| **语音通信**         | 内置语音模块，可连接 PSTN、SIP，支持 IP 电话    |
| **无线控制器**       | 某些型号内嵌 WLAN 控制器                        |
| **应用服务**         | 可部署虚拟机或容器运行特定网络功能或安全服务    |
| **SD-WAN 控制器**    | ISR 可作为 Cisco SD-WAN 的边缘设备（C-edge）    |

---

## 📦 常见型号示例：

| 系列                    | 适用场景          | 说明                           |
| ----------------------- | ----------------- | ------------------------------ |
| **ISR 900 系列**        | 小型办公室或 SOHO | 精简型，性价比高               |
| **ISR 1000 系列**       | 分支机构          | 多端口、支持 LTE 模块          |
| **ISR 4000 系列**       | 中大型分支        | 支持多服务模块、SD-WAN、大吞吐 |
| **ISR G2 系列**（已老） | 早期版本          | 如 ISR 2900/3900 系列          |

---

## 📌 ISR 适用场景举例：

- **银行、零售门店**：用 ISR 同时处理路由、防火墙、POS VPN、视频监控等
- **企业远程分支机构**：部署 ISR 实现远程连接、统一管理
- **语音 + 数据网络整合**：一台设备处理语音呼叫与数据上网

---

## 🧠 ISR 与其他设备的对比：

| 设备类型                     | 说明                                          |
| ---------------------------- | --------------------------------------------- |
| **ISR**                      | 多功能集成平台，适合分支部署                  |
| **ASR**                      | Aggregation Services Router，高性能核心路由器 |
| **Catalyst 路由器（C8000）** | 新一代融合路由交换设备                        |
| **防火墙（FTD/ASA）**        | 专注安全控制，不负责路由                      |
| **Meraki MX**                | 云管理路由器，更适合轻量 SD-WAN 场景          |

---

## ✅ 总结一句话：

> **Cisco ISR 是一个“全能型选手”，特别适合需要同时部署路由、安全、语音、VPN 的分支场景，帮助企业节省设备和管理成本。**
