思科 CASB（Cloud Access Security Broker）是 Cisco 提供的一种云安全解决方案，用来在企业用户和云服务之间**提供安全控制与可视性**。

---

## ✅ 简单来说，Cisco CASB 是干什么的？

> **帮助企业安全地使用 SaaS 应用和云服务。**

---

## 🔍 核心功能：

### 1. **发现（Discovery）**

- 自动识别员工正在使用哪些**云服务（SaaS）**，包括影子 IT（未经批准的服务）。
- 比如检测员工在用 Dropbox、Google Drive、Zoom、ChatGPT 等。

### 2. **可视性（Visibility）**

- 提供对云应用的使用细节：谁在用、用什么设备、做了什么操作（上传/下载/分享）。
- 帮助企业知道数据流向哪里。

### 3. **数据保护（DLP）**

- 检测和防止敏感数据（如身份证号、客户信息）被上传到云平台。
- 与 Cisco Secure Email、Secure Endpoint、Secure Web Gateway 等协同防止数据泄露。

### 4. **访问控制（Access Control）**

- 依据用户、位置、设备等设定访问策略：
  - 允许只读、不允许上传
  - 阻止从海外 IP 访问
  - 对移动设备强制加密

### 5. **威胁防护（Threat Protection）**

- 检测云平台中的恶意行为，比如异常登录、恶意文件上传。
- 联动 Cisco Talos 情报和 SecureX 平台，增强威胁响应能力。

---

## 🚀 Cisco CASB 的平台名：

Cisco 的 CASB 功能整合在其 **Cisco Umbrella + Secure Access + SecureX** 平台中，具体产品名常见包括：

- **Cisco Cloudlock**（CASB 的前身）
- **Cisco Umbrella SIG（Secure Internet Gateway）**
- **Cisco Secure Access（SASE 架构的一部分）**

---

## 🧠 场景举例：

| 场景                               | CASB 能力             |
| ---------------------------------- | --------------------- |
| 员工私自用 Google Drive 传资料     | 自动发现 + 警告或阻止 |
| 防止员工上传客户名单到 ChatGPT     | DLP + 云访问策略      |
| 管理者想知道公司用了哪些 SaaS 应用 | 流量分析 + 报表生成   |
| 检测 Dropbox 中有恶意文件共享      | 威胁检测 + 封锁       |
