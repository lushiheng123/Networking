# asav 简述？

## Cisco Secure Firewall ASA Virtual（ASAv）是一款虚拟化防火墙解决方案，旨在为各种环境提供灵活且一致的安全策略。以下是对其核心特性和优势的总结：

## 🔐 核心功能

- **全面的 VPN 支持**ASAv 提供站点到站点、远程访问和无客户端 VPN 功能，支持 Cisco AnyConnect 客户端，确保员工在任何地点、任何设备上都能安全访问企业网络 

- **统一的安全策略管理**无论是在物理数据中心还是在私有或公共云中部署，ASAv 都能实现一致的安全策略，简化管理并减少配置错误 

- **灵活的部署选项**支持在 VMware、KVM、Hyper-V、OpenStack、AWS、Azure、GCP、OCI 等多种平台上部署，满足不同组织的多样化需求 

---

## 🚀 性能与扩展性

ASAv 提供多种性能级别，适应不同规模的业务需求：

| 许可类型 | 状态检测吞吐量 | 同时连接数 | VPN 对等体数 |
| -------- | -------------- | ---------- | ------------ |
| ASAv5    | 100 Mbps       | 50,000     | 50           |
| ASAv10   | 1 Gbps         | 100,000    | 250          |
| ASAv30   | 2 Gbps         | 500,000    | 750          |
| ASAv50   | 10 Gbps        | 2,000,000  | 10,000       |
| ASAv100  | 20 Gbps        | 4,000,000  | 20,000       |
| ASAv-U   | 高达 90 Gbps   | 8,000,000  | 30,000       |

这些性能指标确保 ASAv 能够满足从小型办公室到大型企业的数据处理和安全需。

ASAv 支持在主要的云平台上部署，包括 AWS、Azure、GCP 和 OC 在这些平台上，ASAv 提供与本地部署相同的安全功能，确保在云环境中的应用和数据同样受到保。
