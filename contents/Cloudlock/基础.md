思科 **Cloudlock** 是一款 **云访问安全代理（CASB，Cloud Access Security Broker）**，用于保护企业在使用 **SaaS、PaaS 和 IaaS 云服务**时的数据安全和合规性。

---

### 🔐 Cloudlock 是做什么的？

简单来说，它帮助企业 **在不影响用户体验的前提下，对云服务中的敏感数据、用户行为、第三方应用进行可视化和控制**。

---

### 🌩️ Cloudlock 的核心功能：

| 功能模块                  | 描述                                                                                 |
| ------------------------- | ------------------------------------------------------------------------------------ |
| **数据泄露防护（DLP）**   | 识别云应用中的敏感数据（如身份证号、信用卡号、源代码等），并设置自动防护策略。       |
| **用户行为分析（UEBA）**  | 检测异常用户行为，如账户被盗、异常登录等。                                           |
| **第三方应用控制**        | 分析用户授权的第三方云应用（比如 Google Workspace 中连接的插件），并可设置限制策略。 |
| **多云支持**              | 支持 Google Workspace、Office 365、Salesforce、Box、Dropbox、Slack、AWS 等云服务。   |
| **基于 API 的无代理部署** | 不需要安装代理设备，直接通过云平台 API 进行集成，部署快速、轻量。                    |

---

### 📌 举个例子：

比如你公司使用 Google Workspace 和 Salesforce，Cloudlock 可以：

- 检测员工是否在 Google Drive 上传了包含客户身份证号的 Excel 表格；
- 识别员工是否登录异常，比如深夜从国外登录；
- 控制某些不受信任的第三方插件访问 Google 邮箱。

---

### ✅ 总结一句话：

> **Cisco Cloudlock 是一款 API 驱动的 CASB，用于发现并保护企业在云平台中的敏感数据和用户活动。**
