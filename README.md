# 1. 在原来有 git 配置文件的文件夹中 git branch 检查当前分支，一般是 main

```sh
git init

git checkout -b example

git add .和 git status 常规进行

git commit -m "exampleeee"设置任务

git remote add origin git@github.com:lushiheng123/Networking.git 设置远程连接

git push -u origin example 选择用什么分支上传
```
