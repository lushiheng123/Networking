<h1 align = "center">运维背调</h1>

```sh
git init
git checkout -b desktop
git add .
git commit -m "first commit"
git remote add origin git@github.com:lushiheng123/Networking.git
git branch
git push -u origin desktop
```

```sh
git init
git remote add origin git@github.com:lushiheng123/Networking.git
git remote -v
git fetch origin
git branch -r
git pull origin desktop
```
